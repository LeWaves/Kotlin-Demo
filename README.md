
【Kotlin android demo】

## **描述**
 
这是根据google提供的Kotlin开发语言写的一个android demo 

# 特点：
  类型推断、多范式支持、可空性表达、扩展函数、DSL 支持等。另外对于安卓开发还提供了 Kotlin 安卓扩展和 Anko 库。
